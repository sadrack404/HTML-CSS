Video

Esse é um video amv de one piece

Para por esse video usamos umas tag < video > com os argumentos src que é o diretorio do video, width que é a largura do video controls.

Pode ser escrito da seguinte maneira

< src="../midia/One-Piece.webm" width="500px" controls >

Pode-se escrever da seguinte maneira tambem.

Escrevermos a tag video, mas não passamos o argumento src!, colocamos apenas os argumentos controls autoplay loop width

E dentro da tag video, passamos uma tag soucer onde colocamos o diretorio do video o argumento type.
