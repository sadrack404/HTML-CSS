Principais formatações

Negrito / Destaque

Nesta frase temos um termo em destaque usamos a tag < STRONG > (semântica).

Nessa frase temos um termo em negrito usando a tag < B > (não semântica)

Não use a tag < B >!
Use < STRONG >


Itálico / Ênfase

Nesta frase temos um termo com ênfase usando a tag < EM > (semântica).

Nesta frase temos um termo em itálico usando a tag < i > (não semântica).

Não use mais a tag < i >!
Use < EM >


Texto Marcado

prodemos criar tambem um texto marcado usando a tag < MARK >.

E no outro parágrafo, temos outro texto marcado no final.


Texto Grande e Pequeno

Para escrever um texto grande usamos a tag < big > porem, não se eutiliza mais essa tag pois está fora dos padrões do html 5

Para escrever um texto pequeno usamos a tag < small >

Estamos criando um texto grande e um texto pequeno nesse parágrafo.


Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Para isso utilizamos a tag < del >!


Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Para isso utilizamos a tag < ins >!
