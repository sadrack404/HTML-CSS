Código Fonte / Pré-formatação

O código abaixo foi escrito com a tag < pre >

Essa tag exibe o texto que está dentro dela, da maneira em que é escrita, isso é, pulando as linhas ou mostrando os espaçoes em branco da exata maneira em que foi escrita.

Dentro da tag < pre > colocamos uma tag < code > ela estiliza um trecho de código, da maneira que é escrito!

                }
            }
